# Lexicon AI — User Guide

Welcome to Lexicon AI, your intelligent research workspace. This guide will help you master the platform's core features.

## 1. Workspaces & Collections
- **Workspaces**: Isolated hubs for specific research projects. Each has its own chat and documents.
- **Collections**: Folders to group related workspaces (e.g., "Finance", "Legal").

## 2. Document Analysis
- **Uploading**: Drag and drop PDFs or Markdown files onto the chat area.
- **Context-Aware Chat**: Once uploaded, the AI can answer questions specifically about those documents.
- **Metadata**: Lexicon automatically extracts Author and Title information.

## 3. Multi-Agent Workflows
When you ask a complex question, Lexicon orchestrates multiple agents:
- **Researcher**: Finds facts and extracts data.
- **Analyst**: Synthesizes and critiques the findings.
- **Writer**: Compiles the final response.

## 4. Research Tools (Plugins)
- **ArXiv**: Search academic papers.
- **Wikipedia**: Fetch general knowledge.
- **Code Interpreter**: Run Python code for math/data logic.

## 5. Sharing
Toggle a workspace to **Public** to generate a read-only share link for collaborators.

## 6. Keyboard Shortcuts
- `CMD/CTRL + K`: Open Command Palette.
- `ENTER`: Send message.
- `SHIFT + ENTER`: New line.
