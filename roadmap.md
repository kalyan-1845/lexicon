# Lexicon AI — Strategic Roadmap

## ✅ Phase 1: The Foundation (Completed)
- [x] **Core RAG Engine**: PDF uploading and context-aware chat.
- [x] **AI Summarization**: Automatic document indexing and overview generation.
- [x] **Markdown Insights**: Integrated note-taking system for research.

## ✅ Phase 2: Professional Evolution (Completed)
- [x] **Pro Minimalist Design**: High-performance neutral theme for focused research.
- [x] **Workspace Isolation**: Real-time state persistence for separate research hubs.
- [x] **Collection Hierarchy**: Folder-based organization for workspaces.

## ✅ Phase 3: Intelligence & Interactivity (Completed)
- [x] **Multi-Agent Logic**: Parallel task execution for Researcher/Analyst orchestration.
- [x] **Plugin Ecosystem**: Modular extensions for ArXiv, Wikipedia, and Code Execution.
- [x] **Local LLM Integration**: Full support for Ollama execution.

## ✅ Phase 4: Enterprise Scale (Completed)
- [x] **Mobile Responsive UI**: Layout optimization for mobile devices (PR #44).
- [x] **Security Infrastructure**: Security Policy and Advisory tracking (PR #40, #65).
- [x] **Developer Experience**: Contributing guidelines and Type safety (PR #42, #57).
- [x] **Data Portability**: Export to JSON and Markdown (PR #60, #69).

## 🚀 Phase 5: Production Hardening (In Progress)
- [ ] **Auth 2.0**: OAuth integration and RBAC (Role Based Access Control).
- [ ] **Vector Store Sync**: Persistent pgvector storage for RAG context.
- [ ] **Collaborative Notes**: Real-time CRDT-based note editing.

## 🔮 Future Scope
- **AI Browser Extension**: Clip research directly from the web into Lexicon.
- **Lexicon API**: Public endpoints for third-party integrations.
