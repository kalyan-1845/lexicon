# Security Policy

## Supported Versions

We actively support security updates for the following versions of Lexicon AI:

| Version | Supported |
|---------|-----------|
| 2.x.x   | ✅ Yes    |
| 1.x.x   | ❌ No     |

## Reporting a Vulnerability

We take the security of Lexicon AI seriously. If you find any vulnerability or security concern, please do not disclose it publicly. Instead, follow these steps:

1. Draft a **GitHub Security Advisory** directly in the [Repository Advisories tab](https://github.com/kalyan-1845/lexicon/security/advisories).
2. Alternatively, contact the maintainer Kalyan Reddy (@kalyan-1845) directly on GitHub.

### Our Response Process

- **Acknowledgment**: You will receive an acknowledgment of your report within **48 hours**.
- **Assessment**: We will perform a detailed security assessment within **7 days** of acknowledgment and keep you updated on the progress.
- **Remediation**: If the vulnerability is validated, we will work on a patch and release a new version.
- **Disclosure**: Public disclosure will be coordinated once the patch is successfully deployed.

Thank you for helping keep Lexicon AI safe and secure! 🛡️
